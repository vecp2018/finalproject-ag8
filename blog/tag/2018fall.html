在 Github 線上編輯網誌

2018 Fall 起, 結合 Travis CI, 可以讓使用者直接在 Github 倉儲中編輯網誌.

設定步驟

. 從 Travis CI 以 Github 帳號登入後, 設定要使用 Travis CI 的倉儲. . 準備 .travis.yml 與 Makefile

.travis.yml 檔案:

language: python  
python:  
- 3.7-dev
branches:
  only:
  - gh-pages
install:
- pip install pelican Pygments Markdown beautifulsoup4
script:
- make publish html
